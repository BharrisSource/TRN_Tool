Equations & Simulation Models

Numerical Aperture & Acceptance

Numerical Aperture
NA = √(n₁² − n₂²)
Defines how much light the fiber can accept.
Acceptance Angle
θₐ = arcsin(NA)
Maximum external launch angle for guided light.
Critical Angle
θc = arcsin(n₂ / n₁)
Minimum internal angle for total internal reflection.

V-number & Modes

Normalized Frequency
V = (2πa / λ) · NA
Determines how many modes the fiber supports.
Mode Condition
V < 2.405 → single-mode
Only one mode exists below this threshold.
Modes Estimate
Step: M ≈ V² / 2
Graded: M ≈ V² / 4
Approximate number of supported modes.

Dispersion

Modal Dispersion
Step-index actual: t(θ) = n₁L / (c₀ cosθ), so Δt = tmax − tmin
Worst-case step-index: ΔTs ≈ Ln₁Δ / c₀
Graded-index: ΔTg ≅ Ln₁Δ² / (8c₀)
Actual step-index spread is calculated from the currently launched angle range. Worst-case uses the full-fiber textbook formula. Graded-index uses the provided graded-fiber approximation rather than the inaccurate constant-index ray-delay model. Single-mode uses 0 modal dispersion.
Chromatic Dispersion
Δt = D · Δλ · L
D in ps/(nm·km), Δλ in nm, L in km.
Total Broadening
Δt = √(Δt_modal² + Δt_chromatic²)
Combined pulse spreading.

Bandwidth

Bandwidth–Distance
BW = BWD · Lref / L
Bandwidth decreases with length.
Bit Rate Approximation
Bitrate ≈ Bandwidth
Used as a simple estimate in the simulator.

Simulation Models

Single-mode Coupling
η ≈ (1 − θ / θₐ)¹·³
Approximation used to model coupling efficiency vs launch angle.
Multimode Signals
Total = Σ individual modes
Each mode arrives at a different time; the envelope is their sum.
Visualization Note
Normalized display scale
Pulse shapes are scaled for visibility; numeric values remain physically based.
Source presets
LED → larger Δλ, Laser → smaller Δλ
The source buttons automatically set spectral width to typical teaching values.
